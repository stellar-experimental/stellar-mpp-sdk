Quick Start (Terminal)

Run these two commands in separate terminals. Replace the keys with your own testnet keys.

# Set your Stellar public key as the payment recipient STELLAR_RECIPIENT=GAYF33NNNMI2Z6VNRFXQ64D4ISKL2PFOR3GDUM4H4PTJ63HAJBCZKA2N \ npx tsx examples/charge-server.ts
# Set your Stellar secret key to sign the payment STELLAR_SECRET=SCZANGBA5YHTNYVVV3C7CAZMCLMXPXBDM42SZGFDJHSIA7FGXVCPTCGZ \ npx tsx examples/charge-client.ts
# Start server in background, wait, then run client STELLAR_RECIPIENT=GYOUR_PUBLIC_KEY npx tsx examples/charge-server.ts & sleep 2 STELLAR_SECRET=SYOUR_SECRET_KEY npx tsx examples/charge-client.ts

Need testnet keys? Create them at Stellar Laboratory and fund with Friendbot.
